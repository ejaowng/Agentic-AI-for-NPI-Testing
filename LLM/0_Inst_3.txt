// MOs list:

ExternalGNBCUCPFunction 	// represents an external Central Unit – Control Plane (CU-CP) function that your node needs to communicate with.

TermPointToGNodeB		// represents a logical termination point in your node’s configuration which connects it to another gNodeB

RiPort 				// represents a specific radio interface port or endpoint associated with a node’s radio interface
TnPort 				// Transport Network Port — it represents a logical or physical port on the transport network side

VlanPort 			// represents a logical VLAN (Virtual Local Area Network) interface or sub-interface on a network element
SectorEquipmentFunction 	// represents the configuration and management of the radio sector equipment within a 5G base station or vRAN node.
FieldReplaceableUnit		// represent to instance of Field Replaceable Unit MO representing a remote radio unit or vDU (virtual Distributed Unit)
